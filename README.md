# FileLinks - 专业级网站同步管理与链接地图工具

`FileLinks` 是一个基于 Python 的自动化工具，旨在为个人网站提供“工程化”的同步与管理方案。它通过智能状态追踪、多线程传输和直观的 UI，将原本繁琐的 FTP/SFTP 同步和链接更新过程变得无比高效。

## 🌟 核心特性

- **🔄 智能同步引擎**：
    - **双端状态追踪**：通过隐藏的 `.sync_state` 文件记录本地与云端的同步状态。
    - **冲突检测**：自动识别本地与云端同时修改的冲突并发出警告，防止代码覆盖。
    - **增量更新**：基于文件大小、修改时间和 MD5 校验和，仅传输变化部分。
- **🚀 并发加速传输**：支持多线程并发上传/下载（并发数可调），显著提升包含大量小文件的网页项目的同步速度。
- **⚙️ 交互式配置管理**：
    - **主机配置向导**：内置交互式表单，轻松管理每个项目的 `server.json`。
    - **域名配置**：支持从 `CNAME` 文件动态获取并管理访问域名。
- **🔗 智能链接地图**：一键生成结构化的 `link.md`，支持按扩展名自定义图标，自动忽略无关文件。
- **🎨 工业级 UI 界面**：
    - **项目看板**：进入项目即刻预览主机、路径、域名等核心配置。
    - **全量预览树**：同步前展示完整的项目结构，通过颜色（红/黄/绿）直观呈现变动。
- **🔒 安全与自动化**：
    - **自动忽略**：启动项目时自动在后台完善 `.gitignore`，确保敏感信息（如 `server.json`）不被泄露。
    - **操作日志**：自动记录同步历史到 `.sync_log`。

## 🚀 快速开始

### 前提条件

确保已安装 [uv](https://github.com/astral-sh/uv)。

### 运行步骤

1. 克隆或下载本项目。
2. 在项目根目录下运行：

   ```bash
   uv run main.py
   ```

3. **初次进入项目**：输入或拖入您的网站项目路径。程序将自动为您创建或更新忽略配置文件。

## ⚙️ 配置说明

### 1. 全局配置 (`config.json`)

控制工具的全局行为：

```json
{
  "max_workers": 4,      // 最大并发传输线程数
  "max_history": 16,	//  项目历史保存数量
  "ignore": [".gitignore", ".surgeignore"],
  "icons": {
    ".html": "🌐",
    ".css": "🎨",
    ".js": "📜",
    ".png": "🖼️",
    "default": "📄"
  }
}
```

### 2. 项目配置管理

您不再需要手动编辑 JSON 文件，直接在工具菜单中操作：

*   **选项 4 (⚙️ 主机配置)**：配置 FTP/SFTP 的主机、端口、账户及路径。
*   **选项 5 (🌐 域名配置)**：设置该项目对应的访问域名（自动同步至 `CNAME` 文件）。

## 📖 操作指南

1.  **🔄 智能同步**：最推荐的日常操作。自动判断双端差异，执行增量更新。
2.  **📤 强制上传**：以本地为准。适合本地大修后，清空云端并重推整个项目。
3.  **📥 强制下载**：以云端为准。适合在另一台设备上初次拉取项目，或回滚本地改动。
4.  **🔗 生成链接**：根据当前项目结构与域名，更新 `link.md` 地图。

## 🛠️ 技术细节

- **同步状态文件 (`.sync_state`)**：存储于项目根目录及服务器端，用于增量对比。
- **忽略机制**：工具会自动将 `server.json`, `.sync_state`, `.sync_log` 等文件物理加入忽略列表，确保安全。
- **并发机制**：使用 Python `concurrent.futures` 实现。
